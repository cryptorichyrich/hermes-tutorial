## What is Hermes Agent?

Hermes Agent is a 24/7 autonomous AI Agent that learns from experience. It remembers your preferences, creates reusable skills, and gets better every time you use it. Connect via Telegram, Discord, Slack, or 14 other platforms. MIT licensed, self-hosted, runs on a $5 VPS.
